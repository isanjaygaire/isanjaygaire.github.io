Stocker Web Application Mockup

Stocker is an application to track stock markets and let the users select the stock to find more information on the selected stocks. As well as a reminder and warning message to the user if the stocks is the changing frequently. Users will be able to browse the stock from graphic, but cannot make transaction of stocks. User can contact the stock broker to make the purchase of a stock. Stock manager will be able to track multiple stock to see the changes in graphics with warning through queries.
